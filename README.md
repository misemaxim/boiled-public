# Boiled - Cartography Notepad

**Boiled - Cartography Notepad** is designed to provide a quick and easy way to visualize your data on a map and share it with others, while giving you the ability to customize how that data is represented. The project is not built with the intention of managing large geospatial datasets; however, it allows you to micro‑manage your data, run scripts on it through plugins to detect specific conditions, automatically update your map based on defined rules, and more. Overall, it offers a sandbox‑style environment where you can build your own custom maps.

**Boiled - Cartography Notepad** is a web application that can be deployed locally or on your own server. You can expose it to others with separate access points for administrators - who can manage and configure data - and public users, who can view up‑to‑date information that has been shared with them.

This project is **open‑source**, and you are welcome to build your own solutions within the terms of the license and its limitations.
